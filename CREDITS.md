# Credits

Sophie's Escape: The Witch's Castle is built on several open-source libraries and third-party services. Every direct dependency and analytics service is listed here. Audio assets will be added to this file once picks are made at the v0.2 audio build.

This file is the attribution record for the project. It is cross-referenced from the README and will be linked from the in-game Credits screen when that screen is built.

## Audio assets

Audio assets will be listed here once the v0.2 audio build is complete. The team evaluates CC0 and Creative Commons BY libraries such as [freesound.org](https://freesound.org/) and Web Audio Application Programming Interface procedural synthesis alongside the BBC Sound Effects Library (bbcrewind.co.uk/sound-effects). Each audio file picked will appear here with its source URL, licence, and attribution line. See also the editor's note in `docs/design-brief.md`, section 5.

## Fonts

No custom fonts are bundled. The game uses a system font stack (Roboto > Segoe UI > Helvetica Neue > sans-serif for the user interface; Georgia > Times New Roman > serif for display text pending Tim's confirmation). No font licence attribution is required.

## Icons

No icon set is bundled in v0.1. Lucide Icons is under evaluation for v0.2 (MIT Licence, see [Lucide on GitHub](https://github.com/lucide-icons/lucide)).

## 3D assets

No third-party geometry, texture, or model assets are used in v0.1. All 3D geometry in the Dungeon Cell placeholder room is generated procedurally via Three.js primitives. Third-party 3D assets, if added in later builds, will be listed here.
